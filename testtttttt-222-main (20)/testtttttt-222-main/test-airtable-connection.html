Airtable Connection Test - FREMKYS

What this test does:
  • Sends a test order to your edge function
  • Shows detailed logs from the submission
  • Helps identify why orders aren't appearing in Airtable

Step 1: Prepare

  1. Open your browser's Developer Tools (Press F12)
  2. Go to the "Console" tab
  3. Keep it open to see detailed logs

Step 2: After Test

  1. Check Console logs (F12) for detailed information
  2. Check Supabase logs: Edge Function Logs
  3. Check Airtable: Orders Table
