# دليل نظام الروابط المباشرة للمنتجات

## نظرة عامة

تم إضافة نظام روابط مباشرة لكل منتج يسمح بمشاركة روابط مخصصة لأي منتج، بحيث عند فتح الرابط يتم توجيه المستخدم مباشرة إلى صفحة المنتج.

---

## كيف يعمل النظام؟

### 1. عند النقر على منتج
عندما يضغط المستخدم على أي منتج:
- يتغير URL في المتصفح إلى: `yoursite.com/?id=product-1`
- يتم عرض صفحة تفاصيل المنتج
- يمكن نسخ الرابط من شريط العنوان

### 2. عند فتح رابط مباشر
عندما يفتح شخص رابط منتج مباشرة:
- يتم قراءة معرف المنتج من URL
- يتم عرض صفحة المنتج مباشرة
- لا حاجة للتنقل عبر الموقع

### 3. عند الضغط على زر الرجوع
عندما يضغط المستخدم على زر الرجوع في المتصفح:
- يعود إلى الصفحة الرئيسية
- التاريخ في المتصفح يعمل بشكل طبيعي

---

## أمثلة على الروابط

### صيغة الرابط:
```
https://yoursite.com/?id=123
```
أو
```
https://yoursite.com/?product=123
```

### أمثلة واقعية:
```
https://fremkys.com/?id=product-1
https://fremkys.com/?id=product-2
https://fremkys.com/?product=product-1
```

## المزايا

1. **مشاركة سهلة**: يمكن نسخ الرابط ومشاركته مباشرة
2. **وسائل التواصل**: الروابط تعمل على WhatsApp، Facebook، Instagram
3. **SEO**: تحسين محركات البحث
4. **تجربة أفضل**: المستخدم يصل مباشرة للمنتج المطلوب

## كيفية الاستخدام

### للزوار:
1. اضغط على أي منتج
2. انسخ الرابط من شريط العنوان
3. شارك الرابط مع أي شخص

### للتسويق:
1. افتح أي منتج في الموقع
2. انسخ الرابط
3. استخدمه في:
   - منشورات Instagram
   - رسائل WhatsApp
   - إعلانات Facebook
   - Stories

---

## التقنيات المستخدمة

تم استخدام:
- **URLSearchParams**: لقراءة وكتابة معاملات URL
- **window.history.pushState**: لتغيير URL بدون إعادة تحميل الصفحة
- **popstate event**: للتعامل مع أزرار الرجوع والتقدم
- **useEffect hook**: لقراءة URL عند تحميل الصفحة

لا توجد مكتبات خارجية مطلوبة!

---

## ملاحظات مهمة

1. **يعمل بدون مكتبات إضافية**: النظام يستخدم فقط Web APIs القياسية
2. **متوافق مع جميع المتصفحات**: يعمل على Chrome، Firefox، Safari، Edge
3. **آمن**: لا توجد مخاطر أمنية
4. **سريع**: لا يؤثر على أداء الموقع
