## Field Mapping Changes

### New Fields Added to Airtable Payload

| Field Name | Type | Description |
|------------|------|-------------|
| `Items` | JSON String | Complete items array with all product details |
| `Order Number` | String | Unique order identifier (FREMKYS-timestamp) |
| `Shipping Address` | String | Full shipping address (was abbreviated before) |
| `Total` | Number | Total amount (subtotal + shipping) |

### Fields That Are NOT Sent (Auto-generated by Airtable)

| Field Name | Why Not Sent |
|------------|--------------|
| `ID` | Auto-incremented by Airtable |
| `Full Customer Name` | Calculated by Airtable formula |
| `Confirmation Date` | Set by Airtable formula/automation |
| `Created At` | Auto-generated timestamp |
| `Updated At` | Auto-generated timestamp |
